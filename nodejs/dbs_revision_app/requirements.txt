1. Install the package modules with "npm install".
2. Create a .env file at root folder, insert "TOKEN_KEY = secret".
3. To run, open up a terminal and run "npm start".
4. Open up a browser and go to http://localhost:3000.
5. Use Postman app to execute the CRUD operation.

# SQL
1. Install "MySQL Connector/ODBC".
2. mysql -u root -p
3. SQL queries:
    SHOW DATABASE;
